# Food Web App

This is an in-progress web app for tracking recipes.

## Installing dependencies

To install dependencies in both the backend and frontend, you can run `npm i` here in the root.

## Development server

To serve both the backend and the frontend, run `npm run dev` here in the root.

The backend API will be served on port 3000 and UI will be served on port 4200.

Visit http://localhost:4200 to view the UI.

## Unique features!

Some of the features that make this recipe app unique:

- Can minimize/maximize recipes to taskbar
- Can run timers from recipes
- Weekly planner for meals
- Recipe filtering
- Complex UI fully responsive on mobile/small screens (styling still WIP)
- Copy/paste steps/ingredients from other websites (WIP)
- Smooth, animated UI (TBA)
